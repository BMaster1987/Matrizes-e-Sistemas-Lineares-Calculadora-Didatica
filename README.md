# Calculadora Profissional de Matrizes + Apostilas de Apoio

## Português

### Visão geral
Este repositório reúne um **HTML offline de apoio didático** e **três apostilas em PDF** para estudo de:

- matrizes e operações com matrizes;
- determinantes;
- matriz inversa;
- sistemas lineares.

O objetivo do material é oferecer uma experiência de estudo mais visual, prática e organizada, unindo **teoria**, **exemplos resolvidos**, **exercícios** e uma **calculadora didática em HTML** para acompanhamento passo a passo.

Este projeto foi desenvolvido para **uso educacional**, com apoio de **Inteligência Artificial**, como complemento das apostilas e das aulas sobre Álgebra Linear elementar.

---

### Arquivos do projeto

#### 1. `calculadora_matrizes_profissional(1).html`
Aplicação HTML offline voltada ao estudo de Álgebra Linear elementar.

A calculadora foi feita com foco em clareza visual e uso didático. Ela permite trabalhar, entre outros tópicos, com:

- multiplicação de matrizes;
- determinantes;
- cofatores e menores complementares;
- matriz inversa;
- sistemas lineares.

Também apresenta resolução organizada e, em vários casos, **passo a passo detalhado**, o que a torna útil para estudo individual, revisão e uso em sala de aula.

#### 2. `Apostila_matrizes_operações_prof_Alessandro_parte01.pdf`
Apostila introdutória sobre matrizes, incluindo definição, tipos, leitura de elementos, operações e exercícios com resolução comentada.

#### 3. `Apostila_determinantes_inversa_prof_Alessandro_parte02.pdf`
Apostila de continuação com foco em determinantes e matriz inversa, incluindo teoria, métodos de cálculo, exemplos resolvidos, exercícios e gabarito comentado.

#### 4. `Apostila_sistemas_lineares_prof_Alessandro_parte03.pdf`
Apostila dedicada a sistemas lineares, com classificação, métodos de resolução, forma matricial, exercícios e resoluções detalhadas.

---

### Como usar o HTML

#### Requisitos
- Navegador atualizado (Google Chrome, Opera, Microsoft Edge, Firefox ou similar).
- Não é necessário instalar nada.
- O arquivo pode ser usado **localmente e offline**.

#### Passo a passo
1. Baixe ou copie o arquivo `calculadora_matrizes_profissional(1).html`.
2. Abra o arquivo diretamente no navegador.
3. Escolha a aba do conteúdo que deseja estudar.
4. Defina a ordem da matriz ou as dimensões necessárias.
5. Preencha os valores numéricos nos campos indicados.
6. Execute o cálculo na própria interface.
7. Analise o resultado e o passo a passo exibido.
8. Quando disponível, utilize a exportação em imagem para registrar a resolução.

#### Sugestão de uso pedagógico
Uma forma eficiente de estudar com este material é:

1. ler a teoria na apostila correspondente;
2. acompanhar alguns exemplos resolvidos;
3. testar os mesmos exemplos no HTML;
4. resolver novos exercícios sem consulta;
5. usar a calculadora para conferir o procedimento e revisar os passos.

## English

### Overview
This repository brings together an **offline educational HTML tool** and **three PDF handouts** for studying:

- matrices and matrix operations;
- determinants;
- inverse matrices;
- linear systems.

The goal of this material is to provide a more visual, practical, and organized learning experience by combining **theory**, **worked examples**, **exercises**, and a **didactic HTML calculator** with step-by-step support.

This project was developed for **educational use**, with the support of **Artificial Intelligence**, as a companion resource for the handouts and lessons on elementary Linear Algebra.

---

### Project files

#### 1. `calculadora_matrizes_profissional(1).html`
An offline HTML application designed for studying elementary Linear Algebra.

The calculator was created with a strong focus on visual clarity and educational use. It supports topics such as:

- matrix multiplication;
- determinants;
- cofactors and minors;
- inverse matrices;
- linear systems.

It also provides an organized solution layout and, in many cases, a **detailed step-by-step explanation**, making it useful for self-study, review, and classroom support.

#### 2. `Apostila_matrizes_operações_prof_Alessandro_parte01.pdf`
Introductory handout on matrices, including definitions, types of matrices, reading matrix entries, operations, and exercises with commented solutions.

#### 3. `Apostila_determinantes_inversa_prof_Alessandro_parte02.pdf`
Continuation handout focused on determinants and inverse matrices, including theory, calculation methods, worked examples, exercises, and commented answer keys.

#### 4. `Apostila_sistemas_lineares_prof_Alessandro_parte03.pdf`
Handout dedicated to linear systems, including classification, solution methods, matrix form, exercises, and detailed solutions.

---

### How to use the HTML file

#### Requirements
- A modern web browser (Google Chrome, Opera, Microsoft Edge, Firefox, or similar).
- No installation is required.
- The file can be used **locally and offline**.

#### Step by step
1. Download or copy the file `calculadora_matrizes_profissional(1).html`.
2. Open the file directly in your browser.
3. Select the tab related to the topic you want to study.
4. Choose the matrix order or required dimensions.
5. Enter the numerical values in the indicated fields.
6. Run the calculation through the interface.
7. Examine the result and the displayed step-by-step solution.
8. When available, use the image export option to save the worked result.

#### Suggested study workflow
A productive way to use this material is:

1. read the theory in the corresponding handout;
2. follow a few worked examples;
3. reproduce the same examples in the HTML tool;
4. solve new exercises on your own;
5. use the calculator to check the procedure and review each step.

### Suggested study order
1. **Part 1:** Matrices and operations.
2. **Part 2:** Determinants and inverse matrices.
3. **Part 3:** Linear systems.
4. **HTML tool:** Continuous use as a support tool for checking results and visualizing procedures.
